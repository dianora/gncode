Koumbia Juillet 2019

Introduction to programming and the Java language

The GNCode pilot session was held in Fria from December 9th to December 28th, 2019 in our training center within the 'Groupe Scolaire Afrique Unie' school, and included students from the Josip Broz Tito, Amílcar Cabral and Christ Roi high schools. The selection was made based on the results of the BEPC 2019, and the retention based on the availability and level of interest of the invited students.

Curriculum

Dates Times Agenda
December 9th - 13th 2019 4PM - 6PM Preliminary preparations
Introduction to the Windows file system; Typping; Text editing and indentation; Introduction to programming via Hour of code minecraft
December 16th - 28th 2019 8AM - 6PM Programmation en Java (CSBridge)
The CSBridge program inspires the next generation of computer scientists by helping young people discover the joy and value of computer programming. It is based on computer science teaching methodologies that have been used at Stanford University for over 20 years. It is organized in different cities and universities around the world and has already been used to teach the basics of programming to thousands of high school students. For more information on the CSBridge program, visit https://guinea.csbridge.org.
January 7th 2020 4PM - 6PM Certificate ceremony
Student presentations; Instructor Presentations; Certificate ceremony.
